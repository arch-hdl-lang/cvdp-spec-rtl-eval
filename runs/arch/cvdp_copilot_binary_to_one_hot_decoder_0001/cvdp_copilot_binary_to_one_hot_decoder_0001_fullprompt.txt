2. **`OUTPUT_WIDTH`**: Defines the width of the output `one_hot_out`, which should have `2<sup>BINARY_WIDTH</sup>` bits if fully utilized. However, `OUTPUT_WIDTH` can be set independently to limit the number of one-hot output bits.`OUTPUT_WIDTH=32`
