Natural-language specification:

Design a System Verilog module named `ttc_counter_lite.sv` that implements a lightweight timer/counter. The module is a timer/counter that supports interval mode operation, interrupt generation, and AXI-Lite register-based control and status monitoring.


## Design Specifications
**Module Name**:
`ttc_counter_lite`

### Inputs:
1. **`clk`** (1 bit): Clock signal for synchronous operation.
2. **`reset`** (1 bit): Active-high reset signal to initialize the timer.
3. **`axi_addr`** (4 bits): Address bus for AXI-Lite read/write operations.
4. **`axi_wdata`** (32 bits): Data bus for AXI-Lite write operations.
5. **`axi_write_en`** (1 bit): Write enable signal for AXI-Lite writes.
6. **`axi_read_en`** (1 bit): Read enable signal for AXI-Lite reads.

### Outputs:
1. **`axi_rdata`** (32 bits): Data bus for AXI-Lite read operations.
2. **`interrupt`** (1 bit): Interrupt signal, asserted when the counter matches the configured match value.

---

## Functional Requirements

### Timer Operation:
1. **Counter Register (`count`):**
   - Counts up on every clock cycle when the timer is enabled.
   - Holds the count value at the match value in non-interval mode.
   - Resets to the reload value upon match in interval mode.
2. **Match Value Register (`match_value`):**
   - Configures the counter match value for interrupt generation.
3. **Reload Value Register (`reload_value`):**
   - Configures the counter reload value for interval mode.

### Modes of Operation:
1. **Enable (`enable`):** Enables or disables the timer operation.
2. **Interval Mode (`interval_mode`):**
   - When enabled, the counter reloads to the configured reload value on a match event.
   - When disabled, the counter holds at the match value.
3. **Interrupt Enable (`interrupt_enable`):**
   - Enables or disables interrupt generation when the counter matches the match value.

### Write Operations:
- **Match Value Register:** Configure the counter match value.
- **Reload Value Register:** Set the counter reload value.
- **Control Register:** Control timer enable, interval mode, and interrupt enable.

### Read Operations:
- **Count Register:** Read the current value of the counter.
- **Match Value Register:** Retrieve the configured match value.
- **Reload Value Register:** Retrieve the configured reload value.
- **Control Register:** Read the current state of control bits (enable, interval mode, interrupt enable).
- **Status Register:** Check the interrupt status and clear the interrupt by writing to this register.

### Counter Logic:
- Increments on every clock cycle when enabled.
- Reloads to the reload value on a match in interval mode.
- Stops incrementing and holds at the match value in non-interval mode.

### Match Detection:
- Compares the counter value to the match value.
- Sets the `match_flag` when a match occurs.

### Interrupt Generation:
- Asserts the `interrupt` signal when `match_flag` is set and interrupts are enabled.
- Clears the `interrupt` signal when the status register is written.

### Reset Behavior:
- Clears all registers, disables the timer, and resets the counter and interrupt signals.
